### Test de Conceptos de Redes – 20 Preguntas

1. **¿Cuántas capas tiene el modelo OSI?**  
   a) 4  
   b) 5  
   c) 7  
   d) 6  

2. **¿Cuál de las siguientes capas del modelo OSI se encarga de la representación de datos para la aplicación?**  
   a) Capa de Sesión  
   b) Capa de Presentación  
   c) Capa de Transporte  
   d) Capa de Red  

3. **En el modelo TCP/IP, ¿qué capa es equivalente a las capas 1 y 2 del modelo OSI?**  
   a) Capa de Transporte  
   b) Capa de Internet  
   c) Capa de Aplicación  
   d) Capa de Acceso a la Red  

4. **¿Qué protocolo de la capa de aplicación traduce nombres de dominio a direcciones IP?**  
   a) DHCP  
   b) DNS  
   c) FTP  
   d) HTTP  

5. **¿Cuál de estos protocolos es orientado a la conexión?**  
   a) UDP  
   b) TCP  
   c) ICMP  
   d) ARP  

6. **¿Qué campo del encabezado TCP se utiliza para indicar el siguiente byte esperado por el receptor?**  
   a) Número de Secuencia  
   b) Número de Acuse de Recibo (ACK)  
   c) Tamaño de Ventana  
   d) Checksum  

7. **¿Qué dispositivo trabaja principalmente con direcciones MAC (capa 2 del modelo OSI)?**  
   a) Router  
   b) Switch  
   c) Firewall  
   d) Hub  

8. **¿Cuál es el rango de puertos bien conocidos?**  
   a) 0-1023  
   b) 1024-49151  
   c) 49152-65535  
   d) 1-1000  

9. **¿Qué protocolo se utiliza para asignar direcciones IP dinámicamente en IPv4?**  
   a) DNS  
   b) DHCPv4  
   c) ARP  
   d) ICMPv4  

10. **¿Qué tipo de mensaje ICMP se utiliza para hacer ping?**  
    a) Tipo 3 – Destino inalcanzable  
    b) Tipo 8 – Solicitud de eco  
    c) Tipo 11 – Tiempo excedido  
    d) Tipo 5 – Redireccionamiento  

11. **¿Qué proceso divide los datos en unidades más pequeñas para su transmisión en la red?**  
    a) Encapsulación  
    b) Segmentación  
    c) Multiplexación  
    d) Desencapsulación  

12. **¿Qué protocolo de la capa de transporte no garantiza la entrega de los datos?**  
    a) TCP  
    b) UDP  
    c) HTTP  
    d) FTP  

13. **¿Qué campo del encabezado IP limita el número de saltos que puede dar un paquete antes de ser descartado?**  
    a) TTL (Time To Live)  
    b) Checksum  
    c) Protocolo  
    d) Flags  

14. **¿Qué mecanismo utiliza TCP para controlar la cantidad de datos enviados antes de recibir un reconocimiento?**  
    a) Temporizadores  
    b) Tamaño de ventana  
    c) Números de secuencia  
    d) Flags de control  

15. **¿Qué protocolo de enrutamiento es un estándar abierto y utiliza el algoritmo de estado de enlace?**  
    a) EIGRP  
    b) BGP  
    c) OSPF  
    d) RIP  

16. **¿Qué significa el flag SYN en el encabezado TCP?**  
    a) Sincronización para iniciar una conexión  
    b) Finalización de conexión  
    c) Datos urgentes  
    d) Reset de conexión  

17. **¿Qué capa del modelo OSI se encarga de la entrega de datos entre dispositivos en diferentes redes?**  
    a) Capa de Transporte  
    b) Capa de Red  
    c) Capa de Enlace de Datos  
    d) Capa de Sesión  

18. **¿Qué protocolo se utiliza para traducir direcciones IP a direcciones MAC?**  
    a) DNS  
    b) DHCP  
    c) ARP  
    d) ICMP

    a) Segmento  
    b) Trama  
    c) Paquete  
    d) Bit
