### **Examen Unidad 3: Enumeración de servicios**

1. **¿Qué herramienta se usa principalmente para el escaneo de puertos y enumeración de servicios?**  
   a) Netcat  
   b) Nmap  
   c) Wireshark  
   d) Traceroute  

2. **¿Qué tipo de escaneo TCP no completa la conexión (three-way handshake)?**  
   a) TCP Connect  
   b) TCP SYN  
   c) TCP ACK  
   d) TCP FIN  

3. **¿Qué comando en Linux permite trazar la ruta de paquetes hacia un destino?**  
   a) ping  
   b) netstat  
   c) traceroute  
   d) ifconfig  

4. **¿Qué técnica envía paquetes ICMP a toda una red para descubrir hosts activos?**  
   a) ARP sweep  
   b) Ping sweep  
   c) Port scanning  
   d) Traceroute  

5. **¿Qué herramienta realiza descubrimiento de hosts mediante ARP de forma pasiva?**  
   a) Netdiscover  
   b) Nmap  
   c) Fping  
   d) Hping  

6. **¿Qué estado de puerto indica que un cortafuegos está bloqueando el acceso?**  
   a) Abierto  
   b) Cerrado  
   c) Filtrado  
   d) Desconocido  

7. **¿Qué parámetro de Nmap permite detectar el sistema operativo del objetivo?**  
   a) -sV  
   b) -O  
   c) -sS  
   d) -p  

8. **¿Qué herramienta se usa para capturar banners de servicios como FTP o SSH?**  
   a) Telnet  
   b) Netcat  
   c) Curl  
   d) Wget  

9. **¿Qué script de Nmap comprueba si un servidor FTP permite login anónimo?**  
   a) ftp-bounce  
   b) ftp-anon  
   c) ftp-syst  
   d) ftp-vuln  

10. **¿Qué herramienta detecta tecnologías web como CMS o frameworks desde terminal?**  
    a) Wappalyzer  
    b) WhatWeb  
    c) Nikto  
    d) Gobuster  

11. **¿Qué técnica consiste en probar nombres de directorios y archivos en un servidor web?**  
    a) Banner grabbing  
    b) Fuzzing web  
    c) Port scanning  
    d) OS detection  

12. **¿Qué herramienta de fuzzing web usa el término “FUZZ” para sustituir palabras de diccionario?**  
    a) Dirb  
    b) Dirbuster  
    c) Wfuzz  
    d) Gobuster  

13. **¿Qué script de Nmap enumera recursos compartidos SMB?**  
    a) smb-os-discovery  
    b) smb-enum-shares  
    c) smb-vuln-ms17-010  
    d) smb-protocols  

14. **¿Qué herramienta escanea vulnerabilidades en servidores web y aplicaciones?**  
    a) Nmap  
    b) Nikto  
    c) Netcat  
    d) Traceroute  

15. **¿Qué herramienta utiliza plantillas YAML para detectar vulnerabilidades automatizadas?**  
    a) OpenVAS  
    b) Nuclei  
    c) Nessus  
    d) Legion  

16. **¿Qué escáner de vulnerabilidades es de código abierto y forma parte de GVM (Greenbone)?**  
    a) Nikto  
    b) OpenVAS  
    c) Nuclei  
    d) Nessus  

17. **¿Qué protocolo usa el puerto 445 por defecto en Windows para compartir archivos?**  
    a) FTP  
    b) SSH  
    c) SMB  
    d) HTTP  

18. **¿Qué comando en Linux enumera usuarios y recursos compartidos en sistemas Windows/Samba?**  
    a) nbtscan  
    b) enum4linux  
    c) smbclient  
    d) rpcclient  

19. **¿Qué técnica de escaneo envía paquetes TCP con los flags FIN, URG y PUSH activados?**  
    a) SYN scan  
    b) ACK scan  
    c) XMAS scan  
    d) NULL scan  

20. **¿Qué herramienta permite enumerar y navegar por recursos compartidos SMB con permisos?**  
    a) Smbmap  
    b) Nbtscan  
    c) Nmap  
    d) Netcat  

---

### **Soluciones**

1. **b) Nmap**  
2. **b) TCP SYN**  
3. **c) traceroute**  
4. **b) Ping sweep**  
5. **a) Netdiscover**  
6. **c) Filtrado**  
7. **b) -O**  
8. **b) Netcat**  
9. **b) ftp-anon**  
10. **b) WhatWeb**  
11. **b) Fuzzing web**  
12. **c) Wfuzz**  
13. **b) smb-enum-shares**  
14. **b) Nikto**  
15. **b) Nuclei**  
16. **b) OpenVAS**  
17. **c) SMB**  
18. **b) enum4linux**  
19. **c) XMAS scan**  
20. **a) Smbmap**
