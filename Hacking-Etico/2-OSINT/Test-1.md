### **Examen Unidad 2: Recopilación de información y OSINT**

1. **¿Qué tipo de inteligencia se obtiene de fuentes públicas como registros o redes sociales?**  
   a) HUMINT  
   b) SIGINT  
   c) OSINT  
   d) GEOINT  

2. **¿Qué buscador tecnológico indexa servicios y puertos activos en lugar de contenido web?**  
   a) Google  
   b) Shodan  
   c) DuckDuckGo  
   d) Bing  

3. **¿Qué herramienta permite navegar de forma anónima mediante enrutamiento cebolla?**  
   a) VPN  
   b) I2P  
   c) TOR  
   d) Proxychains  

4. **¿Qué archivo indica a los buscadores qué páginas no deben indexar?**  
   a) sitemap.xml  
   b) index.html  
   c) robots.txt  
   d) .htaccess  

5. **¿Qué comando se usa en Linux para descargar un sitio web completo desde línea de comandos?**  
   a) curl  
   b) wget  
   c) grep  
   d) dig  

6. **¿Qué herramienta analiza metadatos de archivos en línea de comandos?**  
   a) Metagoofil  
   b) Exiftool  
   c) Sherlock  
   d) Recon-ng  

7. **¿Qué servicio permite consultar información de registro de dominios e IPs?**  
   a) DNS  
   b) Whois  
   c) Netcraft  
   d) Robtex  

8. **¿Qué técnica permite encontrar subdominios probando nombres con un diccionario?**  
   a) DNS spoofing  
   b) Zone transfer  
   c) DNS bruteforcing  
   d) Reverse lookup  

9. **¿Qué herramienta de OSINT busca un nombre de usuario en múltiples redes sociales?**  
   a) Maltego  
   b) Sherlock  
   c) FOCA  
   d) Recon-ng  

10. **¿Qué protocolo permite transferir una copia completa de la zona DNS desde un servidor primario?**  
    a) AXFR  
    b) IXFR  
    c) SOA  
    d) PTR  

11. **¿Qué herramienta visualiza información OSINT mediante entidades y transformadas?**  
    a) FOCA  
    b) Maltego  
    c) Recon-ng  
    d) SpiderFoot  

12. **¿Qué comando en Linux se usa para buscar texto dentro de archivos?**  
    a) find  
    b) locate  
    c) grep  
    d) cat  

13. **¿Qué herramienta automatiza la enumeración de DNS con múltiples técnicas?**  
    a) dnsrecon  
    b) nslookup  
    c) host  
    d) dig  

14. **¿Qué tipo de registro DNS asocia un nombre de dominio a una dirección IP?**  
    a) MX  
    b) CNAME  
    c) A  
    d) TXT  

15. **¿Qué herramienta permite extraer texto de imágenes mediante OCR?**  
    a) Exiftool  
    b) Tesseract  
    c) Metagoofil  
    d) Pdftotext  

16. **¿Qué sitio web permite ver versiones antiguas de páginas web archivadas?**  
    a) Have I Been Pwned  
    b) Wayback Machine  
    c) Shodan  
    d) Dehashed  

17. **¿Qué técnica OSINT se usa para encontrar información en filtraciones de datos?**  
    a) Google dorks  
    b) Reverse image search  
    c) DNS enumeration  
    d) Leak checking  

18. **¿Qué herramienta de línea de comandos realiza consultas DNS detalladas?**  
    a) ping  
    b) traceroute  
    c) dig  
    d) netstat  

19. **¿Qué framework de OSINT está escrito en Python y tiene módulos como usufy y mailfy?**  
    a) Recon-ng  
    b) OSRFramework  
    c) Maltego  
    d) FOCA  

20. **¿Qué técnica permite recuperar texto oculto tras un pixelado inseguro?**  
    a) OCR  
    b) Steganografía  
    c) Depix  
    d) Reverse search  

---

### **Soluciones**

1. **c) OSINT**  
2. **b) Shodan**  
3. **c) TOR**  
4. **c) robots.txt**  
5. **b) wget**  
6. **b) Exiftool**  
7. **b) Whois**  
8. **c) DNS bruteforcing**  
9. **b) Sherlock**  
10. **a) AXFR**  
11. **b) Maltego**  
12. **c) grep**  
13. **a) dnsrecon**  
14. **c) A**  
15. **b) Tesseract**  
16. **b) Wayback Machine**  
17. **d) Leak checking**  
18. **c) dig**  
19. **b) OSRFramework**  
20. **c) Depix**
