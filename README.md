# Documentación de Ciberseguridad

Este repositorio recopila material y guías organizadas por áreas, para facilitar el estudio y consulta de temas clave en seguridad informática.

---

## Contenido del repositorio

- **Analisis Forense Informatico**  
  Guías y pruebas sobre investigación de incidentes, recuperación de evidencia y análisis de sistemas comprometidos.

- **Bastionado Redes y Sistemas**  
  Buenas prácticas para asegurar sistemas, redes y servicios ante ataques.

- **Hacking Etico**  
  Técnicas de evaluación de seguridad con fines educativos y éticos.

- **Incidentes Ciberseguridad**  
  Procedimientos y ejemplos de gestión de incidentes de seguridad.

- **Normativa Ciberseguridad**  
  Legislación, políticas y estándares relevantes en ciberseguridad.

- **Produccion Segura**  
  Recomendaciones para mantener sistemas y servicios en producción seguros.

---

## Navegación

Usa el **menú lateral** para explorar los distintos temas y documentos.  
Cada sección contiene archivos Markdown con instrucciones, guías y ejemplos prácticos.

## Contribuciones

1. Haz un fork de este repositorio.
2. Crea tu rama: `git checkout -b mi-nueva-seccion`
3. Realiza tus cambios y haz commit: `git commit -m "Añadir nueva sección"`
4. Haz push a tu rama: `git push origin mi-nueva-seccion`
